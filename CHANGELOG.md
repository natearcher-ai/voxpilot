# Changelog

## 0.1.0 (2026-02-26)

- Initial release
- On-device voice transcription via Moonshine ASR (Tiny + Base)
- Toggle and push-to-talk modes
- Voice activity detection with configurable sensitivity
- Send to VS Code chat, insert at cursor, or copy to clipboard
- Auto-download models from HuggingFace on first use
- Cross-platform audio capture (Linux/macOS/Windows)
